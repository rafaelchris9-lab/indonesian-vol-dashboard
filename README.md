# indonesian-vol-dashboard
This is my first project, to help trade easier in the Indonesian stock market (Where I reside), and to determine entry to the stock market plus to forecast the expected Volatility using GARCH(p,q) on account of heterogeneity. It is based on the tutorial made by Roman Paolucci in youtube so all credit goes to him, plus some designs and calculation that I learned from stats class. Cheers!

What I used:
1. Language: Python 3.14.4
2. UI: Tkinter
3. Source: Yahoo Finance
4. Data Analysis and Visualization: Numpy, Stats, Pd, Arch, Matplotlib
5. Methods: Descriptive Statistics, Linear Regression, GARCH(p,q), Volatility Regime Analysis

With this project, I hope to make more data science projects to bridge the gap between my college experience and more practical real-world application. And it's just plain old fun. 

NOTE: AI was used for debugging, and creating the color template for the design, plus needed some help on extracting the yh files.
